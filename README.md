# AI 实盘参考系统

这是一个适合 iPhone Safari 使用的期货/指数/CFD 策略训练与实盘参考 PWA。系统不会自动下单，不连接真实账户；它会训练策略、保存排行榜，并通过 LINE 发送实盘操作参考。

## 核心功能

- 训练模式：全品种随机学习并同步 Cloudflare KV。
- 胜率前10排行榜：保存策略、胜率、交易次数、回撤、盈亏比和参数。
- 实盘参考LINE：从排行榜前10中选策略，结合最新行情计算方向、参考入场、止损、止盈/OCO。
- 免费保护模式：打开App不自动访问云端，只有点击按钮才消耗Cloudflare/LINE额度。

## 实盘参考LINE会发送什么

当有明确信号时，LINE会包含：

- 当前建议：做多参考 / 做空参考
- 参考入场价
- 止损价
- 止盈价
- OCO设置：止损 + 止盈
- 失效条件
- 历史胜率、交易笔数、最大回撤、盈亏比
- 策略参数

当没有明确信号时，LINE会明确提示：

- 当前建议：无明确信号，观望
- 操作：不下单，等待下一次提醒，不追单

## 重要实盘规则

- LINE提醒是实盘操作参考，不是保证盈利。
- 下单前必须确认外貨EX CFD实际报价、点差和滑点。
- 下单后必须立刻设置止损或OCO。
- 不允许亏损加仓。
- 单笔风险建议控制在本金的0.3%到0.5%。
- 若LINE提示观望，不要为了交易而开仓。

## Cloudflare KV配置

绑定名必须是：`LEARNING_KV`。

当前项目通过 `wrangler.jsonc` 管理绑定，Cloudflare网页上的绑定添加按钮可能是灰色，这是正常的。

## LINE配置

Cloudflare Pages 环境变量：

- `LINE_CHANNEL_ACCESS_TOKEN`
- `LINE_TO`

配置后重新部署，点击 App 中的“实盘参考LINE”测试。

## 推荐使用流程

1. 点“训练模式”。
2. 等全品种学习完成并云端同步。
3. 查看胜率前10排行榜。
4. 点“实盘参考LINE”。
5. 收到LINE后，只在提示做多/做空且你能立即设置止损/OCO时才考虑手动下单。
6. 如果提示观望，不交易。

## 支持品种

- `ES=F` 标普500期货 ES
- `NQ=F` 纳指期货 NQ
- `YM=F` 道指期货 YM
- `GC=F` 黄金期货 GC
- `CL=F` 原油期货 CL
- `NG=F` 天然气期货 NG
- `^N225` 日经225指数参考数据

## 限制

- 当前行情来自 Yahoo Finance chart 数据，适合参考和模拟，不等同于外貨EX CFD报价。
- 系统不会自动连接外貨EX CFD。
- 回测盈利不代表未来盈利。
- 手机自动点击实盘下单风险很高，不建议使用。
