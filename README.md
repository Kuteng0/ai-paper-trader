# AI 模拟交易系统

这是一个适合 iPhone Safari 使用的期货/指数模拟交易 PWA。它不会连接真实账户，不会实盘下单，只用于模拟交易、记录结果、随机学习和参数优化。

## 支持品种

- `ES=F` 标普500期货 ES
- `NQ=F` 纳指期货 NQ
- `YM=F` 道指期货 YM
- `GC=F` 黄金期货 GC
- `CL=F` 原油期货 CL
- `NG=F` 天然气期货 NG
- `^N225` 日经225指数参考数据

## iPhone 使用方法

1. 用 iPhone Safari 打开 Cloudflare Pages 地址。
2. 点 Safari 分享按钮。
3. 选择“添加到主屏幕”。
4. 以后从主屏幕图标打开。

## 最推荐的日常使用流程

1. 打开 App。
2. 选择周期，建议先用 `15分钟`。
3. 点“随机模拟并升级AI”。
4. 等系统逐个品种完成学习。
5. 查看“操作反馈”，确认每个品种是否完成。
6. 查看“胜率前10排行榜”。
7. 排行榜持续积累，之后你随时打开都可以继续学习。

## 全品种随机学习是什么

点击“随机模拟并升级AI”后，系统会：

1. 自动获取 ES、NQ、黄金、原油、日经、道指、天然气的历史行情。
2. 对每个品种随机生成多组策略参数。
3. 每组参数都进行一次完整模拟交易。
4. 把交易次数足够的结果保存到本机浏览器。
5. 按胜率从高到低更新前10排行榜。
6. 从本轮结果里挑一个综合评分最高的策略作为当前参数。

当前学习是本机浏览器内的策略参数学习，不是连接云端数据库，也不是连接真实交易账户。

## 按钮说明

- 获取行情：只获取当前选中品种的历史K线。
- 运行模拟：用当前参数对当前品种进行一次模拟交易。
- 自主学习：对当前品种做参数优化。
- 扫描推荐：对所有品种做确定性参数优化并推荐一个品种。
- 随机模拟并升级AI：对所有品种做随机参数探索，保存结果并更新排行榜。
- 清空榜单：只清空胜率排行榜，不影响真实账户，因为系统没有真实账户连接。
- 重置模拟：清空当前模拟交易日志，但不会清空排行榜。

## 排行榜怎么看

排行榜按胜率从高到低排序，同时显示：

- 品种
- 周期
- 交易笔数
- 净利润
- 最大回撤
- 盈亏比
- 使用的参数

注意：胜率高不一定最好。实盘前仍要看交易笔数、最大回撤和盈亏比。交易笔数太少的高胜率不可靠。

## 风控参数说明

- 初始资金：模拟账户本金，默认 50,000 日元。
- 单笔风险%：每笔模拟交易最多亏本金的比例，建议 0.3% 到 0.5%。
- 日亏损上限%：当天亏损达到该比例后，系统停止当天新模拟交易。
- 最多持仓K线：超过该时间还没止盈止损，就按超时离场。

## 小资金实盘前门槛

在考虑 5万日币小资金跟单前，建议至少满足：

- 连续模拟 1 个月。
- 交易笔数不少于 30 笔。
- 最大回撤低于 5%。
- 盈亏比大于 1.2。
- 不是只靠一两笔大赚撑起来。
- 真实交易必须设置止损或 OCO。

## 重要限制

- 当前行情来自 Yahoo Finance chart 数据，适合研究和模拟，不适合作为实盘下单依据。
- 系统不连接外貨EX CFD 或任何真实交易账户。
- 回测盈利不代表未来盈利。
- 不建议使用手机自动点击方式进行实盘自动下单。
